A plugin to read and display PHP error logs in the WordPress admin dashboard.

== Description ==

That is a powerful tool with which you can monitor you error logs and clear them if the file becomes too large or you want to start clear.

## Technical specification...

* Designed for both single and multi-site installations
* PHP8 fully compatible

== Installation ==

**0 Day Analytics** can be found and installed via the Plugin menu within WordPress administration (Plugins -> Add New). Alternatively, it can be downloaded from WordPress.org and installed manually...

1. Upload the entire `0-day-analytics` folder to your `wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress administration.

Voila! It's ready to go.
